# Cosine Similarity and Matrix Shapes using NumPy

## Overview
This project demonstrates:
- vector operations
- dot product
- norms
- cosine similarity
- matrix multiplication
- matrix shape validation

using NumPy and Google Colab.

## Why Cosine Similarity Matters
Cosine similarity is widely used in:
- NLP embeddings
- recommendation systems
- semantic search
- facial recognition

## Technologies Used
- Python
- NumPy
- Google Colab

## Formula

cos(theta) = (A · B) / (||A|| ||B||)
